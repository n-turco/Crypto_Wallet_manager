# Crypto Wallet Manager

## Project Overview

Crypto Wallet Manager is a C program that simulates a virtual cryptocurrency wallet. It provides a menu-driven interface to perform various financial operations including adding transactions, viewing transaction history, applying transaction fees, and managing wallet balance.

## Key Features

- Add new transactions (deposits and withdrawals)
- View transaction history
- Apply transaction fees
- Display current wallet balance
- Menu-driven user interface
- Data validation and error handling

## Technical Stack

- C
- Standard C libraries (stdio, stdlib, string manipulation)

## How to Run

1. Ensure GCC or another C compiler is installed
2. Clone the repository and navigate to the project directory
3. Compile the source code:

   ```
   gcc -o crypto_wallet main.c
   ```

4. Run the application:

   ```
   ./crypto_wallet
   ```

5. Follow the on-screen menu prompts to manage your virtual wallet
